Privacidad

Política de privacidad

Última actualización: 20 de mayo de 2026.

Esta política explica cómo InmoRadar trata datos en la extensión de navegador, la web, los formularios de contacto, Premium y los servicios asociados.

Resumen rápido

  • La finalidad principal de InmoRadar es analizar anuncios inmobiliarios públicos en portales compatibles y mostrar una valoración orientativa al usuario.
  • No vendemos datos de usuario ni los usamos para publicidad personalizada, scoring crediticio, prestamos o decisiones con efectos juridicos.
  • La extensión guarda datos localmente en el navegador, como preferencias e inmuebles guardados por el usuario.
  • La extensión puede enviar a InmoRadar datos técnicos del anuncio visible para calcular precio, zona, aparcamiento, costes y señales del inmueble.
  • La web usa analítica opcional solo si el usuario la acepta en el banner de privacidad.

Responsable y contacto

Responsable del tratamiento: InmoRadar. Domicilio a efectos de contacto: Avenida Campillo de San Isidro 7, 28521 Rivas-Vaciamadrid (Madrid), España. Email de privacidad y soporte: hola@inmoradar.app.

Si InmoRadar se explota mediante una sociedad o autónomo con identificación fiscal concreta, esos datos se incorporarán al aviso legal y a esta política.

Extensión de navegador

La extensión InmoRadar funciona sobre páginas públicas de anuncios inmobiliarios en portales compatibles, como Idealista, Fotocasa, Pisos.com y Habitaclia. Su finalidad única es ayudar al usuario a analizar el anuncio antes de contactar.

Datos que puede leer la extensión

  • Contenido visible del anuncio: precio, superficie, habitaciones, descripción, características, planta, ascensor, parking, terraza y otras señales públicas del inmueble.
  • URL, dominio, título de la página y portal inmobiliario para comprobar si la página es compatible.
  • Ubicación del inmueble publicada en el anuncio, como municipio, distrito, barrio, zona o calle si aparece en la página. No se usa GPS del usuario.
  • Datos que el usuario decide guardar, como inmuebles favoritos o preferencias de comparación.

Datos que se guardan localmente

La extensión usa almacenamiento del navegador para conservar configuración, estado de instalación, avisos de versión e inmuebles guardados. Estos datos permanecen en el dispositivo del usuario salvo que el usuario los borre, desinstale la extensión o use una función que implique enviarlos voluntariamente.

Datos que pueden enviarse a InmoRadar

Para calcular resultados, la extensión puede consultar APIs propias de InmoRadar con datos del anuncio, como precio, metros cuadrados, municipio, provincia, distrito, zona, características del inmueble y URL de origen. Estos datos se usan para devolver referencias de mercado, cálculos de aparcamiento, ajustes de confianza y señales orientativas.

Si el usuario solicita recibir un informe por email, se tratará el email introducido y la lista de inmuebles seleccionados para generar y enviar ese informe.

Web, contacto y Premium

  • Formulario de contacto: tratamos nombre, email, tema y mensaje para responder la solicitud.
  • Soporte de instalación o compatibilidad: si el usuario solicita ayuda, tratamos email, página/fuente del formulario, referrer, parámetros UTM y user agent para responder la solicitud y diagnosticar el entorno.
  • Área de clientes: tratamos el email de compra para enviar un enlace temporal de acceso al portal de gestión.
  • Premium y pagos: los pagos se gestionan mediante un proveedor externo. InmoRadar no almacena números completos de tarjeta.
  • Informes por email: si el usuario pide un informe, tratamos el email y los datos de los inmuebles incluidos para enviarlo.

Analítica, cookies y consentimiento

La web usa cookies técnicas necesarias para su funcionamiento y puede usar Google Analytics y Google Tag Manager solo si el usuario acepta la analítica opcional. La analítica ayuda a medir visitas, páginas consultadas y rendimiento de forma agregada.

También usamos analítica propia anónima para medir eventos de navegación y conversión, como página visitada, clic en instalación, solicitud de soporte o inicio de checkout. Esta analítica usa un anonymous_session_id, página, evento, navegador/dispositivo aproximado y parámetros UTM; no guarda IP, emails, datos de pago ni user agent completo en la tabla de eventos.

El usuario puede aceptar, rechazar o cambiar su decisión en cualquier momento.

Proveedores y destinatarios

Podemos usar proveedores que actúan como encargados o prestadores necesarios para operar el servicio:

  • Vercel u otros proveedores de hosting y despliegue web.
  • Supabase u otros proveedores de base de datos y almacenamiento operativo.
  • Lemon Squeezy u otros procesadores de pago para Premium y portal de cliente.
  • Resend, Cloudflare Email Routing u otros proveedores de email para contacto, soporte e informes solicitados.
  • Google Analytics y Google Tag Manager solo para analítica web si el usuario la acepta.
  • Proveedores de distribución de extensiones cuando sean necesarios para instalación, revisión y publicación.

No compartimos datos de usuario con terceros para venderlos, revenderlos, crear perfiles publicitarios ajenos, determinar solvencia crediticia ni ofrecer prestamos.

Base legal

  • Ejecución de un servicio solicitado por el usuario: análisis del anuncio, envío de informes, soporte y gestión de Premium.
  • Interés legítimo: seguridad, prevención de abuso, mantenimiento técnico y mejora básica del producto sin publicidad personalizada.
  • Consentimiento: analítica opcional, comunicaciones o tratamientos que lo requieran.
  • Obligación legal: conservación de información fiscal o contable cuando corresponda.

Conservación

Los datos se conservan durante el tiempo necesario para prestar el servicio, atender solicitudes, mantener seguridad y cumplir obligaciones legales. Los datos guardados localmente por la extensión permanecen en el navegador hasta que el usuario los borre o desinstale la extensión. Los datos de soporte y pago se conservarán durante los plazos necesarios para responsabilidades legales, fiscales o contractuales.

Seguridad

Aplicamos medidas razonables de seguridad técnica y organizativa. Las comunicaciones con la web y APIs se realizan mediante HTTPS. Las credenciales operativas y claves de API no se incluyen en la extensión ni se exponen al usuario.

Derechos de las personas usuarias

El usuario puede solicitar acceso, rectificación, supresión, oposición, limitación del tratamiento y portabilidad escribiendo a hola@inmoradar.app. También puede presentar una reclamación ante la Agencia Española de Protección de Datos si considera que el tratamiento no se ajusta a la normativa aplicable.

Menores

InmoRadar no está dirigido a menores de 14 años y no solicita deliberadamente datos de menores.

Cambios de esta política

Podemos actualizar esta política para reflejar cambios técnicos, legales o de producto. Publicaremos siempre la versión vigente en esta página.
