Política de privacidad

Última actualización: 14 de mayo de 2026.

InmoRadar es una extensión de navegador y una web de apoyo para analizar anuncios inmobiliarios. Esta política explica qué datos se usan y con qué finalidad.

Datos que puede tratar la extensión

  • Texto visible del anuncio inmobiliario para detectar precio, superficie, habitaciones, ubicación aproximada y señales del inmueble.
  • Direcciones habituales configuradas por el usuario, como trabajo o colegio, para preparar cálculos de logística diaria.
  • Inmuebles guardados por el usuario para mostrarlos en una lista comparativa.

Almacenamiento local

En la primera versión, las direcciones habituales y la lista de inmuebles se guardan en `chrome.storage.local`, dentro del navegador del usuario. No se venden datos personales ni se usan para publicidad.

Servicios externos

InmoRadar puede consultar servicios externos de mapas, geocodificación o rutas para calcular ubicaciones aproximadas, transporte o tiempos. Estos servicios pueden recibir la dirección o coordenadas necesarias para responder la consulta.

Pagos

Cuando se active la suscripción Premium, el pago se gestionará mediante un proveedor externo como Lemon Squeezy o Stripe. InmoRadar no almacenará datos completos de tarjeta.

Contacto

Para consultas de privacidad, escribe a hola@inmoradar.app.
