Politica de privacidad

Ultima actualizacion: 20 de mayo de 2026.

Esta politica explica como InmoRadar trata datos en la extension de navegador, la web, los formularios de contacto, Premium y los servicios asociados. La URL publica para Chrome Web Store es https://www.inmoradar.app/privacidad.

Resumen rapido

  • La finalidad principal de InmoRadar es analizar anuncios inmobiliarios publicos en portales compatibles y mostrar una valoracion orientativa al usuario.
  • No vendemos datos de usuario ni los usamos para publicidad personalizada, scoring crediticio, prestamos o decisiones con efectos juridicos.
  • La extension guarda datos localmente en el navegador, como preferencias e inmuebles guardados por el usuario.
  • La extension puede enviar a InmoRadar datos tecnicos del anuncio visible para calcular precio, zona, aparcamiento, costes y senales del inmueble.
  • La web usa analitica opcional solo si el usuario la acepta en el banner de privacidad.

Responsable y contacto

Responsable del tratamiento: InmoRadar. Domicilio a efectos de contacto: Avenida Campillo de San Isidro 7, 28521 Rivas-Vaciamadrid (Madrid), Espana. Email de privacidad y soporte: hola@inmoradar.app.

Si InmoRadar se explota mediante una sociedad o autonomo con identificacion fiscal concreta, esos datos se incorporaran al aviso legal y a esta politica.

Extension de navegador

La extension InmoRadar funciona sobre paginas publicas de anuncios inmobiliarios en portales compatibles, como Idealista, Fotocasa, Pisos.com y Habitaclia. Su finalidad unica es ayudar al usuario a analizar el anuncio antes de contactar.

Datos que puede leer la extension

  • Contenido visible del anuncio: precio, superficie, habitaciones, descripcion, caracteristicas, planta, ascensor, parking, terraza y otras senales publicas del inmueble.
  • URL, dominio, titulo de la pagina y portal inmobiliario para comprobar si la pagina es compatible.
  • Ubicacion del inmueble publicada en el anuncio, como municipio, distrito, barrio, zona o calle si aparece en la pagina. No se usa GPS del usuario.
  • Datos que el usuario decide guardar, como inmuebles favoritos o preferencias de comparacion.

Datos que se guardan localmente

La extension usa almacenamiento del navegador para conservar configuracion, estado de instalacion, avisos de version e inmuebles guardados. Estos datos permanecen en el dispositivo del usuario salvo que el usuario los borre, desinstale la extension o use una funcion que implique enviarlos voluntariamente.

Datos que pueden enviarse a InmoRadar

Para calcular resultados, la extension puede consultar APIs propias de InmoRadar con datos del anuncio, como precio, metros cuadrados, municipio, provincia, distrito, zona, caracteristicas del inmueble y URL de origen. Estos datos se usan para devolver referencias de mercado, calculos de aparcamiento, ajustes de confianza y senales orientativas.

Si el usuario solicita recibir un informe por email, se tratara el email introducido y la lista de inmuebles seleccionados para generar y enviar ese informe.

Web, contacto y Premium

  • Formulario de contacto: tratamos nombre, email, tema y mensaje para responder la solicitud.
  • Area de clientes: tratamos el email de compra para enviar un enlace temporal de acceso al portal de gestion.
  • Premium y pagos: los pagos se gestionan mediante un proveedor externo. InmoRadar no almacena numeros completos de tarjeta.
  • Informes por email: si el usuario pide un informe, tratamos el email y los datos de los inmuebles incluidos para enviarlo.

Analitica, cookies y consentimiento

La web usa cookies tecnicas necesarias para su funcionamiento y puede usar Google Analytics y Google Tag Manager solo si el usuario acepta la analitica opcional. La analitica ayuda a medir visitas, paginas consultadas y rendimiento de forma agregada.

El usuario puede aceptar, rechazar o cambiar su decision en cualquier momento.

Proveedores y destinatarios

Podemos usar proveedores que actuan como encargados o prestadores necesarios para operar el servicio:

  • Vercel u otros proveedores de hosting y despliegue web.
  • Supabase u otros proveedores de base de datos y almacenamiento operativo.
  • Lemon Squeezy u otros procesadores de pago para Premium y portal de cliente.
  • Resend, Cloudflare Email Routing u otros proveedores de email para contacto, soporte e informes solicitados.
  • Google Analytics y Google Tag Manager solo para analitica web si el usuario la acepta.
  • Google y Chrome Web Store para distribucion, revision y publicacion de la extension.

No compartimos datos de usuario con terceros para venderlos, revenderlos, crear perfiles publicitarios ajenos, determinar solvencia crediticia ni ofrecer prestamos.

Base legal

  • Ejecucion de un servicio solicitado por el usuario: analisis del anuncio, envio de informes, soporte y gestion de Premium.
  • Interes legitimo: seguridad, prevencion de abuso, mantenimiento tecnico y mejora basica del producto sin publicidad personalizada.
  • Consentimiento: analitica opcional, comunicaciones o tratamientos que lo requieran.
  • Obligacion legal: conservacion de informacion fiscal o contable cuando corresponda.

Conservacion

Los datos se conservan durante el tiempo necesario para prestar el servicio, atender solicitudes, mantener seguridad y cumplir obligaciones legales. Los datos guardados localmente por la extension permanecen en el navegador hasta que el usuario los borre o desinstale la extension. Los datos de soporte y pago se conservaran durante los plazos necesarios para responsabilidades legales, fiscales o contractuales.

Seguridad

Aplicamos medidas razonables de seguridad tecnica y organizativa. Las comunicaciones con la web y APIs se realizan mediante HTTPS. Las credenciales operativas y claves de API no se incluyen en la extension ni se exponen al usuario.

Derechos de las personas usuarias

El usuario puede solicitar acceso, rectificacion, supresion, oposicion, limitacion del tratamiento y portabilidad escribiendo a hola@inmoradar.app. Tambien puede presentar una reclamacion ante la Agencia Espanola de Proteccion de Datos si considera que el tratamiento no se ajusta a la normativa aplicable.

Menores

InmoRadar no esta dirigido a menores de 14 anos y no solicita deliberadamente datos de menores.

Cambios de esta politica

Podemos actualizar esta politica para reflejar cambios tecnicos, legales o de producto. Publicaremos siempre la version vigente en esta pagina.
