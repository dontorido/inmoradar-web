Privacidad

Política de privacidad

Última actualización: 19 de mayo de 2026.

InmoRadar es una extensión de navegador y una web de apoyo para analizar anuncios inmobiliarios. Esta política explica qué datos se usan y con qué finalidad.

Datos que puede tratar la extensión

  • Texto visible del anuncio inmobiliario para detectar precio, superficie, habitaciones, ubicación aproximada y señales del inmueble.
  • Direcciones habituales configuradas por el usuario, como trabajo o colegio, para preparar cálculos de logística diaria.
  • Inmuebles guardados por el usuario para mostrarlos en una lista comparativa.

Almacenamiento local

Las direcciónes habituales y la lista de inmuebles se guardan en el navegador del usuario. No vendemos datos personales ni los usamos para publicidad.

Servicios externos

InmoRadar puede consultar servicios externos de mapas, geocodificación, rutas o análisis para calcular ubicaciónes aproximadas, transporte, tiempos o contexto del inmueble.

Pagos

Cuando se active Premium, el pago se gestionará mediante un proveedor externo. InmoRadar no almacenará datos completos de tarjeta.

Contacto

Para consultas de privacidad, escribe a hola@inmoradar.app.
