# CTAs comerciales en guías editoriales SEO

## Objetivo

Las guías editoriales captan tráfico informacional de usuarios que todavía están comparando vivienda. El CTA comercial debe convertir esa lectura en uso de InmoRadar sin interrumpir la experiencia: ayudar al usuario a revisar el anuncio que ya está mirando antes de llamar, escribir o visitar.

El objetivo no es prometer una decisión automática ni sustituir visita, tasación, asesoramiento financiero o revisión documental. El CTA debe presentar InmoRadar como una capa de análisis previa.

## Ubicación

Cada guía editorial indexable debe incluir:

- Un CTA intermedio después del primer tercio o alrededor de la mitad del contenido principal.
- Un CTA final cerca del cierre de la guía, antes del bloque de fuente, aviso editorial o footer.

El CTA estándar de navegación o footer puede seguir existiendo, pero no sustituye estos dos bloques editoriales.

## Tono

El texto debe ser prudente, útil y orientado a decisión previa. Variantes recomendadas:

- Analiza el anuncio antes de contactar.
- Comprueba si el precio parece razonable.
- Detecta señales que el anuncio puede no dejar claras.
- Compara mejor antes de llamar o escribir.
- Usa InmoRadar gratis para revisar el anuncio.

La formulación base puede ser:

> ¿Estás buscando vivienda en Idealista, Fotocasa u otros portales? Usa InmoRadar gratis para ver información útil del anuncio que muchas veces no aparece clara: precio por m², señales de riesgo, resumen del inmueble y comparación antes de contactar.

## Menciones a portales

Se permite mencionar Idealista, Fotocasa u otros portales solo de forma descriptiva, como ejemplos de lugares donde el usuario puede estar buscando vivienda.

Cuando el CTA mencione portales concretos, debe incluir una aclaración de independencia, por ejemplo:

> InmoRadar es una herramienta independiente y no está afiliada oficialmente a portales inmobiliarios.

## Prohibido

No usar logos, iconos, colores, assets visuales ni elementos de marca de terceros.

No insinuar afiliación, integración oficial, colaboración, partnership, homologación, certificación ni aprobación por parte de Idealista, Fotocasa u otros portales.

No usar textos como:

- Partner oficial de Idealista.
- Integrado con Fotocasa.
- Aprobado por portales inmobiliarios.
- En colaboración con Idealista o Fotocasa.

No usar pop-ups, bloqueos de lectura, overlays persistentes, urgencia artificial, contadores falsos ni dark patterns.

## Enlace

El CTA debe enlazar a una ruta real y no rota, con esta prioridad:

- URL oficial de Chrome Web Store si está configurada en el código.
- `/premium` si es la ruta activa de conversión.
- `/que-analiza` si no hay instalación estable.

No se debe inventar una URL de Chrome Web Store.
