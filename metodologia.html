Como calcula InmoRadar sus senales.

Esta pagina resume las fuentes, reglas y limites que usa InmoRadar para analizar anuncios inmobiliarios antes de contactar.

Precio por m2

El precio por metro cuadrado del anuncio se calcula dividiendo precio total entre superficie declarada. Despues se compara con la mejor referencia disponible.

Nivel geografico

La referencia puede ser de zona, barrio, distrito, municipio o provincia. Si el dato es municipal, se muestra como referencia municipal.

Confianza

La confianza baja cuando falta ubicacion exacta, la referencia es demasiado amplia o hay pocas fuentes disponibles.

[ 01 ] Fuentes

Datos trazables, no cifras inventadas.

InmoRadar prioriza fuentes oficiales o publicas y deja visibles los caveats cuando la precision no es suficiente.

Mercado

Referencias normalizadas en market_price_sources, MIVAU para venta municipal y SERPAVI para alquiler cuando existe dato.

Entorno

OpenStreetMap/Overpass, datos municipales y senales urbanas para transporte, servicios, calles peatonales y aparcamiento.

Edificio

Catastro y fuentes publicas cuando permiten aportar contexto de direccion, antiguedad o caracteristicas del inmueble.

Formula precio m2

precio_m2 = precio_total / superficie_m2

Es una lectura inicial para comparar anuncios, no una tasacion.

Diferencia vs mercado

Se calcula contra la referencia seleccionada y se comunica como senal orientativa: por debajo, en mercado o por encima.

Parking Difficulty

Combina zona regulada, parkings cercanos, calles peatonales, densidad urbana y perspectiva visitante/residente.

Contenido SEO

Solo se publica indexable cuando hay datos reales, fuente, fecha, canonical, FAQ y calidad suficiente.

[ 02 ] Limites

InmoRadar orienta, no certifica.

Las referencias pueden quedar desactualizadas o no representar todas las zonas de un municipio. La decision final debe apoyarse en visita, documentacion, tasacion profesional y criterio propio.
